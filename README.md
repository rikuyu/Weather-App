# Weather-App

現在の天気を取得することができるアプリです。Redux Thunkがどんなものか勉強するために作成しました。

## デプロイ先

https://rikuyu.github.io/Weather-App/ (Github Pages)

## 使用技術

- React
- Redux
- TypeScript
- Tailwind CSS

## 使い方

1. テキストフィールドに天気を知りたい場所の名前をローマ字で入力する。（例：tokyo）
2. 検索ボタンを押す
